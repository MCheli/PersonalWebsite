PTC Quadcopter

IoT - 3D Printing - Education

Description

The PTC Quadcopter project was a multidisciplinary engineering project that I undertook in 2015 in order to create an exciting IoT demo. It was created so that PTC would have a product that is particularly exciting to younger audiences.

To build it, I used the suite of PTC's product development and IoT tools to design a smart and connected quadcopter from scratch. PTC Creo was used for the design, PTC Mathcad was used for weight, density, and balance calculations, PTC Windchill was used to manage the CAD files and BOM, and finally ThingWorx was used to build an IoT application to show the live data from the quadcopter.

Inside the quadcopter is an Arduino based flight controller, which is hooked up to a 900Mhz analog video transmitter, a 2.4Ghz control receiver, and a 5Ghz flight telemetry transmitter. Multiple antennas were used to ensure failsafe control of the quadcopter at all times. Telemetry information is recieved by a nearby laptop which uses a Java ThingWorx Edge SDK to upload the information to the ThingWorx platform. The ThingWorx mashup creation tool was used to create a near-real time visualization of the data in the form of a webpage that can be viewed from anywhere.
